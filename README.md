# React Vite Frontend Project

This is a React frontend project built using Vite as the build tool and bundler.

## Project Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
   The app will be available at [http://localhost:8000](http://localhost:8000).

## Build for Production

To build the project for production, run:

```bash
npm run build
```

## Project Structure

- `src/` - Source code directory contains components, pages, context, and services.
- `public/` - Static assets like images and favicon.
- `vite.config.js` - Vite configuration including dev server port.

## Usage

- Modify source files in `src/` for app development.
- Run `npm run dev` for local development.
- Run `npm run build` for production build.
- Serve production build with any static server.
